# Tunnel Mastery

Tunnel patterns, checklist, and diagnostics for SSH tunnels — whether you're reaching a local-only proxy on a remote box, bypassing middleware auth, or making a flaky link survive sleep/wake.

## What this covers
- Bare SSH local/remote/dynamic forwards
- Persistent tunnel patterns
- systemd-managed tunnels
- Deterministic failure-first troubleshooting
- Named tunnel topologies and diagnostics
- Known SSH footguns and fixes

## Repo layout
- `docs/` — operational playbooks and checks
- `references/` — compact reference sheets
- `scripts/` — reusable snippets and unit templates
